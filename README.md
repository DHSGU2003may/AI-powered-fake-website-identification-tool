Detects fake/spoofed websites in real-time using AI to prevent phishing and data theft.
🛡️ SiteShield – Browser Security Extension






SiteShield is a lightweight browser extension designed to enhance user safety by protecting against malicious, unsafe, and suspicious websites.

🌟 Key Features

🔒 Detects unsafe & potentially harmful websites
🚫 Blocks access to malicious domains
⚡ Fast, lightweight & easy to use
🧠 Improves browsing security awareness
🧩 Simple extension interface
🌐 Works directly in the browser

🚀 Installation Guide
🔧 Load Extension Manually (Developer Mode)

1️⃣ Open Chrome Browser
2️⃣ Go to chrome://extensions/
3️⃣ Enable Developer Mode
4️⃣ Click Load Unpacked
5️⃣ Select the SiteShield-Extension folder
6️⃣ Extension is now active 🎉
